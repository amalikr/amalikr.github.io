# amalikr.github.io
Benvinguts als Projectes
Aquesta és una web amb menú desplegable, icones Font Awesome, mode clar i fosc, i disseny responsiu per mòbil i ordinador utilitzant Bootstrap 5.3.8. Aquest és el punt de partida per editar tot i fer projectes amb el professor Francesc Pérez.

Sóc Eric de 4tA ESO de l'Institut Pompeu Fabra

En aquesta web veurem projectes realitzats durant 4t d'ESO a la matèria de Digitalització:

Projecte 1: Comparar aliments ecològics i convencionals pel que fa al nivell de nutrients: vitamines, minerals, polifenols i altres substànicies beneficoses.
Projecte 2: 22 dies en 22 peus: és una cursa nàutica on aprenem a sumar vectors, calcular distàncies en GPS i altres activitats
